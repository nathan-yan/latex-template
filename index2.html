P1-21 (odd)


P1
\(6x^2y^2dx + 4x^3ydy = 0, M(x, y) = 6x^2y^2, N(x, y) = 4x^3y \\ \Rightarrow M_y = 12x^2y, N_x = 12x^2y\), the equation is exact.

\(G(x, y) = 2x^2y^3, \phi'(y) = 4x^3y - 6x^2y^2 \\ \Rightarrow \phi(y) = 2x^3y^2 - 2x^2y^3 \\ \Rightarrow F(x, y) = 2x^2y^3 + 2x^3y^2 - 2x^2y^3 = \boxed{2x^3y^2 = c}\)


P5
\((x + y)^2dx + (x + y)^2dy = 0, M(x, y) = (x + y)^2, N(x, y) = (x + y)^2 \\ \Rightarrow M_y = 2x + 2y, N_x = 2y + 2x\), the equation is exact.

\(G(x, y) = y^2x + x^2y + \frac{1}{3}x^2, \phi'(y) = x^2 + 2xy + y^2 - (2xy + x^2) \\ \Rightarrow \phi(y) = \frac{1}{3}y^3 \\ \Rightarrow F(x, y) = \boxed{y^2x + x^2y + \frac{1}{3}x^2 + \frac{1}{3}y^3 = c}\)


P9
\((3x^2 + 2xy + 4y^2)dx + (x^2 + 8xy + 18y)dy = 0, M(x, y) = (3x^2 + 2xy + 4y^2),\\ N(x, y) = (x^2 + 8xy + 18y) \\ \Rightarrow M_y = 2x + 8y, N_x = 2x + 8y\), the equation is exact.

\(G(x, y) = x^3 + yx^2 + 4y^2x, \phi'(y) = (x^2 + 8xy + 18y) - x^2 - 8yx \\ \Rightarrow \phi(y) = 9y^2 \\ \Rightarrow F(x, y) = \boxed{x^3 + yx^2 + 4y^2x + 9y^2 = c}\)


P13
\(\frac{x}{(y^2 + x^2)^{3/2}}dx + \frac{y}{(y^2 + x^2)^{3/2}}dy = 0, \\ \Rightarrow M_y = -3x(x^2 + y^2)^{-5/2}y, N_x = -3x(x ^ 2 + y^2)^{-5/2}y\), the equation is exact.

\(G(x, y) = -\frac{1}{\sqrt{x^2 + y^2}}, \phi'(y) = \frac{y}{(y^2 + x^2)^{3/2}} - \frac{y}{(x^2 + y^2)^{3/2}} = 0 \\ \Rightarrow F(x, y) = \boxed{-\frac{1}{\sqrt{x^2 + y^2}} = c}\)


P17
\((3x^2\cos{xy} - x^3y\sin{xy} + 4x)dx + (8y - x^4\sin{xy})dy = 0 \\ \Rightarrow M_y = (3x^3 - \sin{xy}) - x^3(\sin{xy} + yx\cos{xy}), \\N_x = (3x^3 - \sin{xy}) - x^3(\sin{xy} + yx\cos{xy}\), the equation is exact.

\(G(x, y) = x^2(x \cos{xy} + 2), \phi'(y) = (8y - x^4\sin{xy}) + x^2\sin{xy} \\ \Rightarrow \phi(y) = (x^3 - x)\cos{xy} + 4y^2 \\ \Rightarrow F(x, y) = \boxed{x^2(x \cos{xy} + 2) + (x^3 - x)\cos{xy} + 4y^2 = c}\)
